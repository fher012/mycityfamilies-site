MyCityFamilies Support

This is the official support page for the MyCityFamilies app. If you need help using the app, have noticed incorrect information, or want to share feedback, contact us here.

Email Support View Privacy Policy

Contact

  • Email: support@mycityfamilies.com
  • Response time: We aim to respond within 2 business days.
  • Best for: Account access issues, data corrections, bug reports, and app feedback.

Common Issues

  • Login problems: Confirm you are using the same sign-in method you used when creating your account.
  • Old event details: Send the event or place name and the incorrect detail so we can review it.
  • App performance: Restart the app and make sure you are on the latest version before contacting support.

Help

Trouble signing in

Check that you are using the correct email or sign-in provider. If you still cannot access your account, contact support and include the email address linked to the account.

Missing or outdated event/place information

We rely on source information that can change. If you notice outdated details, missing listings, or an incorrect venue, send us the item name, city, and the correct information so we can review it.

App not loading correctly

Try closing and reopening the app, checking your internet connection, and updating to the latest version. If the issue continues, send a short description of what happens on screen.

Feature requests / feedback

We welcome suggestions that make MyCityFamilies more useful for families. Send us your idea and explain how you would use it in the app.

When contacting support: Please include your device type, app version, and screenshots where relevant.

Privacy

You can review how MyCityFamilies handles personal information on our privacy policy page.
